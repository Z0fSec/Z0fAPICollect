# Z0fAPICollect

一款简单的 API 信息收集与记录 Burp扩展套件。

## ✨ 核心功能

### 🌐 API信息收集

- API参数收集
- API匹配记录
- API自由记录
- API记录导出

## 🚀 快速开始

### 安装

```bash
mvn clean package
```

> ⚠️ 请注意：编译后的 jar 包位于 target/ 目录下
